# clear-kerberods
Detect and Clear Kerberods - By George Bouras

wget https://github.com/SLionB/clear-kerberods/archive/master.zip

unzip master.zip

cd clear-kerberods-master

chmod +x busybox

chmod +x clear

chmod +x detect


## detect for infection
./detect

## clear infection ( note that it reboots the server)
./clear
